1 - Je cherche une station ouverte

Si le marqueur est de couleur verte,
la station est ouverte et contient des velos disponibles

si le marqueur est de couleur orange,
la station est egalement ouverte mais il reste peu de vélos (2 ou moins)

Si le marqueur est de couleur rouge,
la station est fermée

2 - les informations de la station cliqué s'affiche

je clique sur un marqueur pour afficher :
- le nom de la station
- l'adresse de la station
- le nombre de velos disponible
- le nombre de places disponibles

3 - Je réserve un velo

Pour réserver un velo je dois signer dans le cadre blanc indiqué après avoir cliqué sur le bouton de réservation - l'adresse de la station
- le nombre de velos disponible
- le nombre de places disponibles
